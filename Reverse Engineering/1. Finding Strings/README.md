# Finding Strings
Looking for useful strings (here, a
licence key) within a compiled binary.

## Execution
```bash
# Build
make

# Run and analyze
./register

# Examine the output
strings ./register

# Find the key of the form XXXXX-XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
./register XXXXX-XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
```
